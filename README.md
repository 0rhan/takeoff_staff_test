# 1. Установить все пакеты

### `npm i`

# 2. Установить json-server

### `npm install -g json-server`

# 3. Запустить

### `npm start`

### `json-server --port 3002 --watch db.json`

Открыть в браузере [http://localhost:3000](http://localhost:3000).
